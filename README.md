Campus Issue Management System

A modern web application that allows students and communities to report, track, and manage issues across different campus groups such as Hostel, Library, Transport, Sports, and Facilities.

Built as a responsive Single Page Application focused on usability, performance, and real-world problem solving.


Overview

Managing campus issues is often slow and unorganized.
This platform provides a centralized system where users can report problems, monitor progress, and collaborate within specific groups.

The application supports role-based access, real-time feedback, and optimized performance for smooth user experience.
